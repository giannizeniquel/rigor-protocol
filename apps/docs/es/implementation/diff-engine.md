# Motor de Diff

## Propósito

El Motor de Diff realiza comparación basada en grafos entre dos especificaciones, generando un ChangeSet estructurado que clasifica las diferencias como rompedoras o no rompedoras.

## Contenido Esperado

- Algoritmo de comparación de grafos
- Generación y estructura del ChangeSet
- Reglas de clasificación de cambios rompedores
- Diff semántico vs estructural
- Formatos de salida (JSON, YAML)
