# Arquitectura General del Sistema

## Propósito

Este documento describe la arquitectura de alto nivel del motor RIGOR, incluyendo sus componentes principales, flujos de datos e interacciones de módulos.

## Contenido Esperado

- Diagrama de módulos mostrando componentes core
- Separación de capas (Entrada → Procesamiento → Salida)
- Flujo de datos desde especificación YAML hasta ejecución
- Responsabilidades de componentes e interfaces
- Puntos de extensión y arquitectura de plugins
