# Motor de Restricciones

## Propósito

El Motor de Restricciones evalúa reglas y restricciones internas durante la validación de especificaciones y ejecución runtime, determinando niveles de severidad y cortocircuitando la evaluación.

## Contenido Esperado

- Lógica de evaluación de restricciones
- Niveles de severidad (ERROR, WARNING, INFO)
- Comportamiento de cortocircuito
- Registro y encadenamiento de restricciones
- Soporte para restricciones personalizadas
