# Parser y Loader

## Propósito

El Parser y Loader es responsable de cargar especificaciones YAML, realizar validación inicial de sintaxis y convertir la entrada en una representación intermedia.

## Contenido Esperado

- Parseo de YAML y manejo de errores
- Validación de sintaxis y reporte de errores
- Normalización del formato de entrada
- Soporte para múltiples fuentes de entrada (archivo, stdin, URL)
- Detección y manejo de codificación
