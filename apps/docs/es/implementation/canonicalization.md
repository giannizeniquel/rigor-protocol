# Motor de Canonicalización

## Propósito

El Motor de Canonicalización produce una representación determinista y estable de las especificaciones que habilita hashing consistente, comparación y caché.

## Contenido Esperado

- Reglas de forma canónica
- Algoritmo de hashing estable
- Ordenamiento y serialización determinista
- Generación de rutas canónicas
- Derivación de claves de caché
