# Motor de Migraciones

## Propósito

El Motor de Migraciones aplica transformaciones atómicas a instancias de proceso persistidas cuando las especificaciones evolucionan, asegurando integridad de datos durante actualizaciones de versión.

## Contenido Esperado

- Lógica de aplicación de ChangeSet
- Manejo de migración atómica vs parcial
- Garantías de idempotencia
- Mecanismos de rollback
- Estrategias de migración offline y perezosa
