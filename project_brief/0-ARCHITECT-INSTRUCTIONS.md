## Your Deliverables

### 1. Setup Script (CRITICAL - Do This First)

Create `setup.sh` — an executable bash script that:
- Creates complete monorepo structure
- Generates all config files from templates
- Installs dependencies
- Sets up Astro and VitePress apps
- Copies design tokens

**This script will be executed by builders to create the base project.**

Example structure:

```bash
#!/bin/bash
set -e

echo "🏗️  RIGOR — Project Setup"
echo "Generated by: Gemini Architect"
echo ""

# Step 1: Create directory structure
mkdir -p rigor/{apps/{web,docs},packages/design-tokens}
cd rigor

# Step 2: Create root package.json
cat > package.json << 'EOF'
{
  "name": "rigor-monorepo",
  "version": "0.1.0",
  "private": true,
  "workspaces": ["apps/*", "packages/*"],
  "scripts": {
    "dev": "turbo run dev",
    "build": "turbo run build"
  }
}
EOF

# ... continue with all setup steps
```

**Include in setup.sh:**
- Directory creation
- All package.json files (root, web, docs, tokens)
- All config files (astro.config.mjs, turbo.json, etc.)
- Dependency installation
- File copying from project-brief/4-assets/
